# MERN Todo App

A simple full-stack to-do list application built using the MERN stack.

##  Features
- CRUD Operations:** Create, Read, Update, and Delete tasks.
- Task Status:** Mark tasks as pending or complete.
- Persistent Storage:** Data is stored in MongoDB.
- Responsive UI:** Built with React and [Tailwind CSS/Bootstrap].

##  Tech Stack
- Frontend:** React, React Hooks, Axios
- Backend:** Node.js, Express
- Database:** MongoDB, Mongoose

##  How to Run Locally
1. Clone the repository: `git clone <your-repo-link>`
2. Install dependencies: `npm install`
3. Set up `.env` file with `MONGO_URI`
4. Run backend: `npm run server`
5. Run frontend: `npm start`
